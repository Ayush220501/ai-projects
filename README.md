# Bulldozer Price Prediction

This project predicts bulldozer auction prices using machine learning.

## Project Structure
- notebooks/ - Jupyter notebook(s) with EDA and modeling
- data/ - Dataset files (optional)
- src/ - Python scripts for preprocessing or model training (optional)
- requirements.txt - Python dependencies

## How to Run
1. Clone the repository:
   git clone https://github.com/Ayush220501/machine-learning-projects.git
2. Install dependencies:
   pip install -r requirements.txt
3. Open the notebook in notebooks/ folder
